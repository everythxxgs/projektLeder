you have to have node installed

and install yarn
`npm install --global yarn`

first run 
`npm install`
in backend folder

then
`yarn`
in frontend folder


to start run in backend folder
-- `npm run start`
then run 
-- `yarn dev`
in frontend folder


for documentation go to 
localhost:3000/api-docs

for frontend go to 
http://localhost:5173/
